# 📚 Kitap Ajanı

Kişisel ruh haline göre yapay zekâ destekli kitap öneri sistemi

# 🚀 Proje Hakkında

Kitap Ajanı, kullanıcıların ruh hali, okuma tercihleri ve son okudukları kitaplara göre kişiselleştirilmiş kitap önerileri sunan bir yapay zekâ destekli sistemdir.

Bu sistem yalnızca öneri sunmaz; aynı zamanda neden bu kitapların önerildiğini açıklayarak şeffaf ve anlaşılır bir deneyim sağlar.

🎯 Amaç
•Kullanıcıya kişiselleştirilmiş kitap önerileri sunmak

•Yapay zekâ ile önerilerin gerekçesini açıklamak

•Öneri sürecini anlaşılır ve yorumlanabilir hale getirmek

🔄 Kullanıcı Akışı
1.Kullanıcı kitap türünü seçer
2.Ruh halini girer veya seçer
3.Son okuduğu kitabı belirtir
4."AI ile Öner" butonuna basar
5.Sistem öneri + açıklama sunar

🧩 Sistem Mimarisi
1. Profil Ajanı
•Kullanıcının ruh hali, tür tercihi ve okuma geçmişini analiz eder
•Kişisel profil oluşturur
2. Öneri Ajanı
•Tür ve ruh hali eşleşmesi yapar
•Tekrar eden önerileri engeller
•Çeşitli ve uygun kitaplar önerir
3. Açıklama Ajanı
•Yapay zekâ (LLM) kullanarak önerilerin nedenini açıklar

🖥️ Özellikler
🎭 Ruh haline göre öneri
🔍 Filtreleme (tür, alt tür, mood)
⭐ Favorilere ekleme
🧾 Açıklamalı öneri sistemi
💻 Modern web arayüzü

🛠️ Kullanılan Teknolojiler
•Python → Backend & API
•JSON → Kitap veri tabanı
•LLM (Groq) → Açıklama üretimi
•HTML / CSS / JavaScript → Frontend
•Ajan Mimarisi → Profil, öneri ve açıklama sistemleri
